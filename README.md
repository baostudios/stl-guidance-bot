# stl-guidance-bot

a dumb project made by a very dumb person.

# setup & installation

clone from https://github.com/thekevinlab/stl-guidance-bot.git
cd into directory

```
$ echo "" > .env
```

inside .env, paste in the following chunk of text

``` 
TOKEN=<bot token>
ID=<bot id>
# leave blank for global commands
SERVER=<server id>
MONGO=<mongo uri>
```

```
$ npm ci
$ npm run build
```
